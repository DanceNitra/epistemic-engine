The Law

  
Grounding-Coupling Law


  

    
"Self-consistency is not truth; only external grounding couples them."

    

      A system builds confidence from internal consistency (consensus, precision, narrow confidence intervals). 
      That confidence tracks accuracy only in proportion to its external‑grounding coupling g. 
      As g falls, the confidence–accuracy gap widens — pumped by internal effort, not by truth.
    

    

      
g = 0.4

r = 0.25

Usage

  
⚡ Commands

  
Everything you need in one CLI. Zero learning curve.


  

    
epistemic claim "..."
Evaluate any claim — Crucible, Grounding, Bridges, Percolation

    
epistemic bridge A B
Find structural isomorphism between two domains

    
epistemic label "..."
Full epistemic verdict — all 5 detectors in one output

    
epistemic percolation X
Check domain phase + criticality %

    
epistemic bridges X
List all bridges for a domain

    
epistemic status
Full engine dashboard — crucible, bridges, domains, vault sync

    
epistemic sync-vault
Live read from your Obsidian vault

    
epistemic report --html
Generate standalone HTML dashboard

    
epistemic claim --json
JSON output for CI/CD pipelines
